# Backend Auth

Backend de autenticación desarrollado con **Node.js** y **Express**. Provee endpoints para registro, login y gestión de sesiones/tokens de usuarios.

## Tecnologías

- Node.js
- Express

## Instalación

```bash
git clone https://github.com/AgustinRodriguez23/Backend-auth.git
cd Backend-auth
npm install
```

## Uso

```bash
npm start
```
